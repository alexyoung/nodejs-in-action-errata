Node.js in Action, Second Edition Errata

Thank you for purchasing Node.js in Action, Second Edition. Please post any errors, other than those listed below, in the book's Author Online Forum. We'll compile a comprehensive list and publish it here for everyone's convenience. Thank you!

Last updated 2017/11/02

  • Page 102, Section 5.9

    Loopback has changed since the time of publishing. If you're using Loopback 3.0, install loopback-cli with npm instead of the strongloop package. The lb tool is now known as slc.

  • Page 109, Section 6.1.1

    Express is no-longer built with Connect since version 4.0.

  • Page 119, Figure 6.6

    The screenshot shows jade when it should show pug.

  • Page 173, Snippet

    The snippet near the end of the page has a Pug style keyword that needs a fullstop to work correctly:

      style.
    h1 {
      font-size: 6em;
      color: #9DFF0C;
    }
  • Page 185, Listing 8.3

    The function is missing two arguments on line 16. It should be: (err, result) => {
